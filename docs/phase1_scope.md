# NOSIK Hub Phase 1 Scope

## Product goal

Build the first usable version of NOSIK Hub: a home tablet dashboard that coordinates the household.

NOSIK is not just a calendar. It connects people, tasks, reminders, groceries, bills, care access, and urgent household alerts.

---

## Core Phase 1 screens

1. Today dashboard
2. Alerts
3. Calendar
4. Tasks
5. Groceries
6. Money pressure
7. NOSIK Pass
8. Care profile
9. Gift Ideas / Santa List
10. Vault
11. Emergency
12. Settings

---

## NOSIK alert levels

### Nudge
Soft reminder.

Example: `Please bring washing in.`

### Knock
Important household alert.

Example: `Bins need to go out before dark.`

### Blast
Urgent full-screen alert.

Example: `DOG IS OUTSIDE — CHECK GATE NOW.`

Blast takes over the Hub and requires someone to accept it.

---

## Access modes

### Parent mode
Full app access.

### Kid mode
Simple job list, grocery add, gift ideas, and ask Mum/Dad.

### Grandparent mode
Limited family events, announcements, and shared gift ideas.

### Carer mode
Care profile, routine, sensory notes, safe foods, emergency details, and handover notes.

### Visitor mode
Basic tonight/today information and emergency access.

---

## What is deliberately not included yet

- Real phone push notifications
- Supabase live sync
- Hardware manufacturing
- Payment/subscription system
- Mortgage, Debt, and Pet standalone modules
- Full document upload/storage
- True secure PIN hashing

These are Phase 2+ items.
