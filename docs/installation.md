# NOSIK Hub Installation Guide

## Run on a computer

1. Unzip the package.
2. Open `hub/index.html` in Chrome, Edge, Safari, or Firefox.

## Run as a local web server

From the package folder:

```bash
python -m http.server 8080
```

Open:

```text
http://localhost:8080/hub/
```

## Run on a tablet

### Android tablet

1. Copy the folder to a local web server or host it online.
2. Open the Hub URL in Chrome.
3. Tap menu > Add to Home screen.
4. Open from the home screen for an app-like display.
5. Enable screen wake/kiosk mode in Android settings or use a kiosk app.

### iPad

1. Host the folder online or on your local network.
2. Open the Hub URL in Safari.
3. Tap Share > Add to Home Screen.
4. Launch from the home screen.
5. Use Guided Access if you want it locked to NOSIK Hub.

## Recommended physical setup

- 10 to 15 inch tablet
- kitchen, hallway, office, or near fridge
- permanent power cable
- simple wall mount or counter stand
- always-on/kiosk mode

## Demo data reset

Open Settings and tap **Reset Demo Data**.
